The following fonts are offered as Settings options but are **NOT bundled**
(they render only if installed on the user's system), so no license text is
shipped for them:

- **TheSansMono Condensed**, **Antithesis** — *free for personal use only*,
  which is **not** compatible with redistributing them inside this publicly
  distributed repository. They appear in the Settings picker tagged "install";
  install them locally to use them.

("Corbi Mono S", which some font sites list, is actually Libertinus Mono
re-labeled — same copyright holder — so it is not offered separately; use the
bundled Libertinus Mono.)
